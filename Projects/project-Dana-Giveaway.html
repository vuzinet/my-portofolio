// OBJECTIVE

This project is a simulation of "Dana Phishing" attacks, created purely for (educational and awareness purposes). It demonstrates how fake giveaway or login pages can be used by attackers to trick users. The goal is to help people (understand, recognize, and defend against phishing attempts), not to perform malicious activities. All simulations run locally in Termux and do not transmit any real user data.
